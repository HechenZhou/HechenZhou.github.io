# Hechen Zhou Personal Academic Homepage

This is a lightweight academic homepage for GitHub Pages.

## Files
- `index.html` — main page
- `assets/style.css` — styles
- `assets/script.js` — mobile menu
- `assets/profile.jpg` — your formal portrait photo

## How to add your photo
1. Prepare a square or nearly square photo.
2. Rename it as `profile.jpg`.
3. Put it into the `assets` folder.
4. Refresh the page. The photo will automatically replace the `HZ` placeholder.

## How to publish on GitHub Pages
1. Create a GitHub repository named `yourusername.github.io`
2. Upload all files in this folder to the repository root
3. Go to `Settings > Pages`
4. Set the source branch to `main` and folder to `/root`
5. Save and wait for deployment
